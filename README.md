## 概述

免费SSR账号，持续更新，免费ssr-subscribed订阅地址。

本项目基于已有的免费SSR账号生成可用的ssr-subscribe地址，用户可直接使用免费的subscribe

使用：

ssr客户端配置 subscribe feed 填写 本项目地址，例如： http://ssr.kanter.cn/dxxzst 即可使用

若觉得本项目对您有所帮助，欢迎start
